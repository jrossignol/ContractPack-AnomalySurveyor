# Contract Pack: Anomaly Surveyor

A contract pack for [Kerbal Space Program](https://www.kerbalspaceprogram.com/) that guides players to discover the game's easter eggs through a series of survey and investigation contracts.

Requires [Contract Configurator](https://forum.kerbalspaceprogram.com/topic/91625-1) and [Module Manager](https://forum.kerbalspaceprogram.com/topic/50533-1).

## Features

- Contracts for every stock anomaly: Mun arches, monoliths, the island airfield, pyramids, Duna face, and more
- A multi-part monolith storyline that builds across the solar system, culminating at Jool
- SCANsat integration — anomaly scanning contracts when SCANsat is installed
- Spoiler-safe design — locations are revealed progressively as you complete contracts
- Compatible with KSP 1.8.x through 1.12.x

## Installation

Install via [CKAN](https://github.com/KSP-CKAN/CKAN) (recommended), or manually copy the `GameData/ContractPacks/AnomalySurveyor` folder into your KSP `GameData/ContractPacks/` directory.

## Recommended Mods

- [WaypointManager](https://forum.kerbalspaceprogram.com/topic/194876-112x-waypoint-manager-new-dependency-added/) — in-flight waypoint navigation
- [SCANsat](https://forum.kerbalspaceprogram.com/topic/72679-1125-scansat-v211-real-scanning-real-science-at-warp-speed-september-20-2025/) — adds anomaly scanning contracts
